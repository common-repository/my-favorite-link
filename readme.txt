Es un Plugin que permite agregar nuestros links favoritos en el dashboard de WordPress, una herramienta de ayuda para los copywriters

== Description ==

Presentamos el plugin "My Favorite Links" que nos permitirá agregar links específicos que deseamos recordar para el sitio web o blog que estamos trabajando.
Una herramienta muy util para copywriters, que suelen guardar muchos links de información, pero que a menudo saturan los marcadores de sus navegadores, este plugin les permitirá llevar un orden por cada sitio web que trabajan, especialmente cuando tienen varias temáticas.
También nos sirve para, por ejemplo listar nuestras propias entradas que requieren actualizaciones.

== Installation ==

1. Carga el plugin dentro de la carpeta `/wp-content/plugins/` o instala desde la lista de plugins
2. Activa el plugin desde la opcion 'Plugins' en el menu de WordPress
3. Entra en la nueva opción llamada "My Links" en el menú lateral
4. Dentro podrás agregar categorías y tus links favoritos en el dashboard de WordPress, también podrás eliminarlos.

== Frequently Asked Questions ==

= ¿Cuantas Categorias y Enlaces puedo Crear? =
No hay limite, puedes crear las categorías y enlaces que consideres necesarios.
= ¿Puedo eliminar los enlaces y las categorías? =
Si, puedes eliminar enlaces individualmente y si eliminas una categoría se eliminarán los enlaces que contiene dicha categoría.
